개인 영어 ​​사전 안드로이드 애플리케이션

개요

이 프로젝트는 '개인 영어사전'이라는 안드로이드 애플리케이션입니다. 이를 통해 사용자는 영어 단어와 번역을 찾아보고, 개인 사전에 단어를 추가하고, 퀴즈를 통해 어휘력을 테스트할 수 있습니다.


기능:

  • 사전: 영어 단어를 검색하고 우즈벡어로 번역해 보세요.
  • 개인 사전: 나중에 참조할 수 있도록 단어를 개인 사전에 저장합니다.
  • 퀴즈: "단어 찾기"와 "단어 입력"이라는 두 가지 유형의 퀴즈로 어휘 지식을 테스트하세요.

전제조건

  • 안드로이드 스튜디오
  • 자바 JDK
  • Genymotion 에뮬레이터(선택 사항이지만 성능 향상을 위해 권장됨)

설치

1단계: Android 스튜디오 설치

  1. 다음에서 Android 스튜디오를 다운로드하세요.여기
  2. 설치 파일을 실행하고 설치 마법사를 따르세요.
  3. 설치 위치와 설치할 구성요소를 선택하세요.
  4. 설치를 완료하고 Android Studio를 시작하세요.

2단계: Genymotion 에뮬레이터 설정r

  1. Genymotion을 다음에서 다운로드하세요. 여기
  2. 아직 설치하지 않았다면 Genymotion과 VirtualBox를 설치하세요.
  3. Genymotion을 실행하고 가상 장치를 추가합니다.
  4. Genymotion Manager에서 가상 장치를 시작합니다.

3단계: 애플리케이션 실행

  1. 에뮬레이터가 실행 중인지 확인하세요.
  2. Android Studio에서 '실행' 버튼을 클릭하여 에뮬레이터에서 애플리케이션을 컴파일하고 실행하세요.

SQLite를 사용한 데이터베이스 설정

SQLite는 애플리케이션의 데이터를 관리하는 데 사용됩니다. SQLite를 설정하고 사용하려면 다음 단계를 따르세요.

  1. 데이터베이스 생성: SQLite용 DB 브라우저를 사용하여 새 데이터베이스를 생성합니다.
  2. 테이블 정의: 사전 항목과 사용자별 데이터를 저장하는 테이블을 만듭니다.
  3. SQL 쿼리 실행: DB 브라우저의 "SQL 실행" 탭을 사용하여 SQL 명령을 실행합니다.
