Personal English Dictionary Android Application

Overview

This project is an Android application called "Personal English Dictionary." It allows users to look up English words and their translations, add words to their personal dictionary, and test their vocabulary through quizzes.


Features

  • Dictionary: Look up English words and get their translations in Uzbek.
  • Personal Dictionary: Save words to a personal dictionary for future reference.
  • Quizzes: Test your vocabulary knowledge with two types of quizzes - "Find the Word" and "Type the Word".

Prerequisites

  • Android Studio
  • Java JDK
  • Genymotion Emulator (optional but recommended for better performance)

Installation

Step 1: Install Android Studio

  1. Download Android Studio from here
  2. Run the installation file and follow the Setup Wizard.
  3. Choose the installation location and components to install.
  4. Complete the installation and start Android Studio.

Step 2: Set Up Genymotion Emulator

  1. Download Genymotion from here
  2. Install Genymotion and VirtualBox if not already installed.
  3. Launch Genymotion and add a virtual device.
  4. Start the virtual device from the Genymotion Manager.

Step 3: Run the Application

  1. Ensure your emulator is running.
  2. Click the "Run" button in Android Studio to compile and run the application on the emulator.

Database Setup with SQLite

SQLite is used for managing the application's data. Follow these steps to set up and use SQLite:

  1. Create a Database: Use the DB Browser for SQLite to create a new database.
  2. Define Tables: Create tables to store dictionary entries and user-specific data.
  3. Execute SQL Queries: Use the "Execute SQL" tab in DB Browser to run SQL commands.
