"Uzbegim" halal market application with web backend

This project was created for online delivery market throughout South Korea. Mainly, customers can order halal products from market named "Uzbegim" online and get their products in within maximum two days


  • For frontend we created both Android and IOS systems in Flutter.

  • Following functions available:
  • - User can register account by just his phone number registred in South Korea. Korean sms sending API integrated
  • - User can easily pick products up into card and pass through checkout
  • - Each user can insert more than one adress for comfortness
  • - We utilized Korean adress API which user just type initial adress or postcode then app show up rest of adress automatically
  • - And so many functions
